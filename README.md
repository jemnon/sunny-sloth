# WOY Frontend

The frontend for the WOY blog. Built using Gatsby, Typescript and Styled Components.

## Dependencies

```
$ npm install -g gatsby-cli
$ npm install -g yarn
```

## Development

```
$ yarn install
$ yarn develop
```
